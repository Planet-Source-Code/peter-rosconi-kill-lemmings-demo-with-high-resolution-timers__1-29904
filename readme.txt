To install the ccrp ActiveX Controls simply move the
 following files to your system folder (usually c:\windows\system\):

	o references/ccrpTmr6.dll
	o references/ccrpTmr6.dep
	o references/ccrpTmr6.tlb
	o references/regsvr32.exe

Then open a dos window (start > run > command)
 then goto the system folder and type:

regsvr32.exe ccrpTmr6.dll
